在iMac中如何配置Hexo

现在有很多IT都喜欢讲自己的blog放在github上。今天查阅各种资料，同时也踩了各种坑，终于搭建自己的Hexo的blog。本文记录一下搭建Hexo的步骤。

##前期准备

Homebrew

Homebrew是Mac系统的包管理工具

注：在安装Homebrew过程中可以遇到权限不足的问题，可以使用命令修改改问题

sudo chown -R `whoami` /usr/local

###Node

brew install node

###注册Github账户并配置SSH
1.注册github：注册说明
2.配置SSH：配置说明

###在github网站上添加一个Repository
需要说明的是该Repository的命名最好是：用户名.github.io

##正式安装Hexo

安装Hexo

npm install -g hexo
hexo init
npm install

在本地创建Hexo文件夹并进入该文件夹

cd ～／Document
mkdir hexo
cd hexo

初始化hexo并在上述文件夹执行生成hexo

hexo init
npm install

测试是否安装成功

hexo server

或者

hexo s

这时在命令行可以看到如下信息

INFO  Start processing
INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.

在浏览器中打开上述网址，看到Hexo本地Demo

##上传到Github

###在本地Hexo文件夹中安装git部署插件

npm install hexo-deployer-git --save

###配置_config.yml

在Hexo文件总找到_config.yml,在该文件的最后deploy，将其改成如下格式：

deploy: 
  type: git 
  repo: https://github.com/用户名/用户名.github.io.git 
  branch: master

##Hexo的使用

添加文件

hexo new “文件名”

生成静态页面

hexo generate

或

hexo g

上传到服务器

hexo deploy

或

hexo d

之后每次执行就可以执行如下三个命令：

hexo clean
hexo generate
hexo deploy

或者起缩写
